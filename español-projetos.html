PROYECTOS:

La asignatura de Métodos y Técnicas Aplicadas al Multilingüismo (MTAM) fue una asignatura llena de interesantes proyectos guiados por el profesor Cláudio Sampaio Menezes, realizados por los alumnos. Aquí hay una recopilación de estos proyectos en Portugués.

Resumen de seminarios

  2. Clasificación automática de documentos
  3. Extracción de textos
  4. Reconocimiento de palabras: transcripción de voz para texto
  5. Resumen automático (PT, EN, FR, SP)
  6. Traducción por computadora
  7. LIBRAS en WebLibras en la computadora
  8. Braille informático
  9. Atom versus Sublime Text
  10. Dreamweaver versus GoLive
  11. Alineamiento Automático por frases
  12. Mozila Komposer Vs. CoffeeCup Free Editor
  13. Notepad++
  14. Diseño de página web
  15. Páginas web monolingües
  16. Páginas web multilingües
  17. Páginas web multimodales
  18. Programación en HTML, CSS y JavaScript
  19. Doity - Plataforma de gestión de eventos
  20. Wordpress, creación de páginas web
